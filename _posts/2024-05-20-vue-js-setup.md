## Node.js 설치

### (선택) yarn 설치

```bash
npm install -g yarn
```

## vue 설치

```bash
npm install -g @vue/cli
```

설치중 오류 시 `yarn`을 이용해 설치한다

```bash
yarn global add @vue/cli
```

## (추가) VSCode 익스텐션 설치

설치 목록
- `Veture`
- `HTML CSS Support`
- `Vue 3 Snippets`

## 프로젝트 생성

```bash
vue create 프로젝트명
```

## 프로젝트 실행

```bash
npm run serve
```
