## 개요

Windows 사용자 기준으로 macOS 사용시 사용하면 좋은 기능을 정리한다

## 주요 키 차이점

- 윈도우에서 `Ctrl`를 통한 단축키 대부분이 `Cmd`에 할당되어있고 `Ctrl`은 거의 사용하지 않는다
- `Enter`는 실행, 폴더 들어가기가 아니라 파일 변경이다.
    - `Cmd` + `down`이 실행 단축키다

## 단축키

- 되돌리기: `Cmd` + `Z` 
- 복사, 붙여넣기: `Cmd` + `C` / `Cmd` + `V`
- 이동하기: `Opt` + `Cmd` + `V`
    - MAC은 `Cmd` + `X` 가 없다
- 지우기 (휴지통): `Cmd` + `Backspace` (윈도우의 `Delete`)
- 지우기 (영구 제거): `Opt` + `Cmd` + `Backspace` (윈도우의 `Shift` + `Delete`)
- 드래그해 복사: 드래그앤드롭
- 드래그해 이동: `Cmd` + 드래그앤드롭
- 드래그해 바로가기 생성: `Opt` + `Cmd` + 드래그앤드롭

## 상단 탭

[이미지 비교샷]

윈도우에서 프로그램을 열때 보이는 *파일, 편집, 보기* 등의 상단 탭의 경우, 윈도우는 프로그램 창의 상단에 있는 반면 맥에서는 모니터 상단에 위치한다.

## Finder 설정

윈도우의 파일 탐색기에 해당하는 앱인데 기본 설정이 상당히 별로다

### 열기 단축키

`Opt` + `Cmd` + `Space`

### 아이콘 위치 정렬하기

[이미지 추가]

Finder에서 `우클릭` > `보기 옵션` 으로 들어간다

`정렬:` 을 `이름`으로 맞춘 다음 맨 아래의 `기본값으로 사용`을 클릭해 지정

### 스크롤바 항상 활성화

[이미지 추가]

스포트라이트(Cmd+Space)에서 `화면 모드` 검색

`스크롤 막대 보기`를 `항상`으로 변경한다

### 폴더 정렬 및 검색 개선

[이미지 추가]

Finder가 포커스된 상태에서 상단탭의 `Finder` > `설정` 에 들어간다

`고급` 에 들어가 `폴더 우선 정렬:`의 `윈도우에서(이름 순으로 정렬 시)`, `데스크탑에서` 둘 다 체크한다

`검색할 때:`를 `이 Mac 검색` 으로 바꾼다

### 보기 개선

- `Opt` + `Cmd` + `P`: Finder 아래 현재 경로를 표시한다
- `Cmd` + `/`: 디렉토리의 남은 용량, 파일 수, 선택한 파일 수 등을 보여준다

단축키를 한번 더 눌러 끌 수 있다

상단탭의 `Finder` > `보기` 에서 지정할 수도 있다
