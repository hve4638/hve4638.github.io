# Pthread

Pthread는 C언어 스레드 라이브러리이다.

## 사용법

### 헤더 include

`<pthread.h>` 를 import하여 사용한다.

### gcc 컴파일 시

기본 라이브러리에 포함되지 않았기 때문에 `-pthread` 옵션을 추가해야 한다.

```bash
gcc -pthread test.c -o test
```
