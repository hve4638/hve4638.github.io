## Node.js 환경 세팅

## Node.js 설치

- [Node.js 설치](https://nodejs.org/en)

```bash
# 설치 확인
npm -v
```

### yarn 설치 (선택)

```bash
npm install -g yarn
```

```bash
# 설치 확인
yarn --version
```

### 프로젝트 생성

```bash
# npm
npm init

# yarn
yarn init
```

명령 실행 후 필요한 정보를 입력하면 `package.json` 파일이 생성된다.
