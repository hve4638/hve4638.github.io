> 실행모드
>
> **:w**							: 저장  
> **:q**							 : 나가기  
> **:set number**		 : 행번호  
> **:set nonumber**	: 행번호 제거  
> **/{x}**						: x 정방향 검색  
> **?{x}**						: x 역방향 검색  
>
> > 편집모드 이동
> >
> > **i**　: 그 자리에서 진입  
> > **a**　: 한칸 이동 후 진입  
> > **o**　: 행 추가후 진입  
> > **s**　: 현재 문자 제거하고 진입  
>
> 


> _이동모드_
>
> hjkl		: 이동  
>
> w, e		: 다음단어 이동  
> b				: 이전단어 이동  
> shift+b	: 이전 문장으로 이동  
>
> 0		: 행의 시작 이동  
> shift+4	: 행의 끝 이동  
>
> shift+h	: 페이지 처음으로 이동  
> shift+l	: 패이지 마지막으로 이동  
>
> gg　　　: 문서 시작으로 이동  
> shift+g　: 문서 끝으로 이동  
>
> x　　　　: 삭제  
> dd　　　: 한행 삭제  
> {X}dd　　: X줄 삭제   
>
> y				: 복사  
> yy　　　 : 한행 복사  
>
> c				: 잘라내기  
> p				: 붙혀넣기   
>
> u				: 되돌리기  
> ctrl+r		: 되돌리기 취소  
>
> v				: 블럭지정  
> shift+v		: 한행 블럭지정  
>
> r				: 리플레이스
> U				: 대문자 전환 (블럭지정)  
> u				: 소문자 전환 (블럭지정)  
> shift+>		: 들여쓰기    (블럭지정)  
>shift+<		: 내어쓰기    (블럭지정)
